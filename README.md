# [FMD] Free Music Downloader.

## No nonsense Music Downloader.

### Download mp3 / opus (360k) using song title and artist.

* supports metadata and thumbnail

```
sh fmd "<query>"
```
### Requirements
* curl
* jq
* wget
* ffmpeg
* seq
* paste
* wc


### Install
```
sudo make install
```

### Uninstall
```
sudo make uninstall
```
